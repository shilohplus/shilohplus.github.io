下学期的选课

发表于 2018-05-31 | 分类于 日常

今天可以看到下个学期的课程了，准确来说应该是下下个学期，可能是因为我一直都不把夏季学期当一个学期啦，主要是我不喜欢这样的小学期安排，而且小学期安排的课都挺水的，对外宣传的时候倒是能说得很好听，我校采取书院制导师制，与国际接轨，一年分为春季学期，夏季学期和秋季学期三个学期，可以缓解大家的课程压力…但是我却觉得那些都只是面子工程，因为夏季学期是从以前的春季学期和秋季学期里挤出来的，所以春秋学期的时间就少了，但课程安排的内容完全没有变少呀（也就是说我们得用更少的时间学同样多的东西…），balabala，扯远了。。不过我这次的夏季学期也没有选任何的课程啦，打算去中山大学的实验室搬砖…

我身边的好多人的学分都够毕业了，可我还差十几分啊…(⊃_⊂)，所以这就意味着我大四也要上好多课呀，主要是因为我大一下和大二上都在计算机系那边混，数学系这边的课我一开始就缺很多，转过来后其实要跟上大家的脚步就有点吃力了，所以导致我还差这么多课，哭唧唧TAT。

下学期我计划是选三到四门课的，因为到时候还会有申请那方面的事情，估计会有点忙，但看了下开出来的课程，选什么课让我有点纠结了…

课程号 课程名 学分 上课时间
CS405 机器学习[中英双语班] 3 1-16周 星期三 1-2节&3-4节
MAT7008 高等统计学（研究生）[中英双语班] 3 1-16周 星期二 7-8节 单周 星期四 5-6节
MAT7035 计算统计（研究生）[中英双语班] 3 1-16周 星期一 7-8节 双周 星期四 1-2节
MAT8011 现代概率论（研究生）[中英双语班] 3 1-16周 星期二 5-6节 双周 星期四 5-6节
FET305 人工智能与博弈论[中英双语班] 3 双周 星期一 1-2节 1-16周 星期五 1-2节
CS303B 人工智能B[英文班（建议选择非计算机专业学生修读）] 3 1-16周 星期三 5-6节&7-8节

哎，真害怕下个学期如果遇到喜欢的课我会不会选个五门..太可怕了，不行，我要克制下，开学选课一时爽，期末就是火葬场啊！！！还是前几个星期观望下，但很多课前两个星期都好水啊。。等过了退课时间以后老师们才会动真格。。。害怕😨

这个暑假还是需要想一想以后的路吧，是想走应用那方面还是想走理论，感觉读phd压力太大了，还是先读个硕士感受下？估计读完硕士屁颠屁颠就去工作赚钱钱了，哈哈哈哈哈哈。

忙活了一下午也没能加上评论区
