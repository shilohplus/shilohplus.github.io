今天买到了番石榴！

发表于 2018-05-27 | 分类于 日常

在食堂水果摊这里等番石榴上新,店里的小哥哥们好像已经习惯我周末中午都会去食堂买番石榴，昨天看到我过去晃悠很久还没买，就啥也没问就跟我说番石榴卖完了。然后有一个小哥哥在整理荔枝，看到我过去就递了我一颗让我尝尝，特别甜的那种，核稍大，肉是白白嫩嫩、水灵的很的那种。想起之前在抽样调查的课上曹敏老师有简单教我们如何辨认荔枝品种，我盯着这颗荔枝瞅了很久，但我还是没能分辨出这个是啥品种，或许是糯米糍？

宿舍大改造！

发表于 2018-05-26 | 分类于 日常

今天天气不错

发表于 2018-05-24 | 分类于 日常

昨天晚上荔园停电了，从晚上7点多停到了晚上12点多，还好舍友提前存了些水，所以回到宿舍以后还能洗澡，要不然我可能得熬夜等水来才会上床睡觉了。我觉得自己在这些方面还是有些洁癖的。

然后就是昨天看了《全职高手特别篇》以后迷的不行，还马上就花了三个多小时简单过了下小说。我觉得自己对一些东西有着很明显的倾向，就是那种，大家一个团队，居住在一起，共同训练，不断朝着某个目标一起努力、前进什么的。所以我对于那些热血运动番真的毫无抵抗力，排球少年或者网球优等生什么的，可能是因为自己真的非常渴望得到这种经历吧，但很可惜呀，一直以来都还没有尝试过呢，不知道以后有没有机会呀。
