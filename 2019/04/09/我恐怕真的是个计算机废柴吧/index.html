我恐怕真的是个计算机废柴吧

发表于 2019-04-09 | 分类于 计算机废柴日常

或许，恐怕这两个字应该是要去掉的。

我的确是个十足的计算机废柴了。我从昨天晚上就在弄如何hexo上加入数学表达式，无论试了多少种教程，最后还是无法在网页上显示出来，我真真真真的要哭了。

而且不止如此，我最近只要修改blog上的东西，GitHub就会给我发page build warning email，我到现在都不懂为什么我需要update my site’s DNS records。然后昨晚上，CNAME文件莫名其妙不见了，然后我重新配置了一次，然后今早上又不见了…然后可能我就一不小心，手滑给输错名字了…再然后我自己也就没管了（因为在我知道我没办法搞定hexo上面的数学公式以后，我写博客的热情就已经被浇灭了一大半了，我电脑里所有写了一半或者是大大小小开了头的文章都已经被宣判死刑了，你们再也没有重见天日的一天了），然后！！！竟然被某人发现我的错误了！！！还给我戳了出来！！！！！从刚才到现在，脑子里都是混乱的，啊啊啊啊啊，这也太丢人了吧，这样不就显得我很蠢吗！！！！！(இдஇ; ) 这样也显得我太蠢了点吧。

而且，我今天晚上啥也没做，就在mac给配置了个LaTeX，就为了能用中文敲LaTeX，虽然我之前的电脑是能够写英文的。然后我在网上找了个教程，emmm，然后就有很多地方我看不太懂，不过巧了的是，那个在简书上写下该教程的人我认识！！我还有微信来着！！！但是对话画风对我来说，简直是惨不忍睹（以下对话重复了差不多两个小时，是我都要崩溃了，感谢对方好人一直到我编译成功为止都没拉黑掉我🙏）。

对方：你把这行填到 sublime text 的 LaTeXTools 的 texpath 里

我：那个…什么叫…把这行填进LaTeXTools的texpath里

对方：/Library/TeX/texbin/tex，这一行

对方：填进 LaTeXTools 设置里面的路径

我：那个…我知道是这一行，但是 LaTeXTools 是什么….

对方：【甩给你一张教程图片】

我：【尝试操作，结果并没有找到，回甩一张没找到该项目的图片】

对方：试试reset

我：哇，真的可以了哎

综上所述，搞懂计算机，我这辈子应该都是做不到的。

我真的很想撬开那些计算机很厉害的人的脑子研究一下，脑回路和我的哪里不一样了呢。
