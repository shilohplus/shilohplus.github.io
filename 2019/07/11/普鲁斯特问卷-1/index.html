普鲁斯特问卷-2019

Jul 11, 2019

又到了回答这份问卷的时候了，我现在还记得去年回答这份问卷的时候自己的状态，焦虑、害怕、不安、痛苦、自闭、毫无起色地挣扎，但也还有，老师带我去吃汉堡吃冰淇淋的那份感动，学长带我们去打羽毛球的那份开心，帮学长看论文改论文听他们讲心得体会的感悟…

嗯，经历过的事都绝对不会忘记的，无论是开心的事还是难过的事，无论是做对了的事还是后悔的事，都不会忘记的。

1.你认为最完美的快乐是怎样的？

大家都毫无隔阂地生活在一起吧。

很羡慕三狮里大家的状态，在一个温馨的老房子。

出了房子大家能有自己的生活，在房子里共同生活的时候彼此之间能够互相得到治愈与救赎。

2.你最希望拥有哪种才华？

能听懂世间所有语言的才华？？

（喂，这更像超能力吧，真的不算偏题吗！）

3.你最恐惧的是什么？

折磨人的疾病，我很怕这些东西，所以最讨厌的地方是医院。

4.你目前的心境怎样？

平静+轻松，还有不少压力呢，暑假的项目一个人做起来有点难度呀。

虽然生活并没有比想象中好很多，但是也渐渐开始学会和自己和解，开始逐渐肯定自己了。

5.还在世的人中你最钦佩的是谁？

王老师。

今年还是王老师呀，哈哈，只要他在世我就会一直钦佩他的。

今年大哥哥就没有当选了，因为我发现我只是羡慕他呀，并不能算钦佩吧。

6.你认为自己最伟大的成就是什么？

成就？？感觉活到现在22年了，也没做出任何成就啊。

不行，我要争取在接下来几年的问卷中能够做出回答在这个问题下的东西，fighting！

7.你自己的哪个特点让你最觉得痛恨？

拖延！！！！

绝对是拖延症！！！！！！！

拖延的时候会导致内耗严重，啥也没做都会觉得特别累。

这一点不改变的话我的人生现在就已经能看到天花板了。

8.你最喜欢的旅行是哪一次？

没有，今年依然没有。

9.你最痛恨别人的什么特点？

自我表现过于明显。

我是真的没办法和这样的人相处，一般遇到这样的人我就会闭嘴啥也不说了。

10.你最珍惜的财产是什么？

年龄！！！！

我要珍惜现在我如花似玉的妙龄啊！！！！！！

变成30岁的大妈了可怎么办啊，呜呜呜

11.你最奢侈的是什么？

最奢侈？？

豪华自助？？？

因为我吃得很少，而且还不怎么爱吃海鲜。

12.你认为程度最浅的痛苦是什么？

我不知道，我现在觉得没有什么痛苦是最浅的，痛苦都是难受的，连最浅的也不能被允许啊。

13.你认为哪种美德是被过高的评估的？

父母的爱。

不做过多评价。

14.你最喜欢的职业是什么？

Data Scientist！

本职工作，最喜欢了～

15.你对自己的外表哪一点不满意？

脸型脸型脸型！！脸太大了！

眼睛眼睛眼睛！！眼睛太小了！

鼻子鼻子鼻子！！鼻子太大了！

16.你最后悔的事情是什么？

大一大二没有好好学习。

但其实我并不后悔，是我自己犯下的错误，我现在的努力都是为了去弥补我曾经做错的那两年。

我也挺感谢那段时光的，没有那个阶段，我一定不会成为现在的自己，也不会找到自己想要做的事。

17.还在世的人中你最鄙视的是谁？

没有。自己也没有任何资格去鄙视别人。

18.你最喜欢男性身上的什么品质？

温柔+上进

19.你使用过的最多的单词或者是词语是什么？

没印象，我今年说话很少，自己都不知道自己经常说啥。

20.你最喜欢女性身上的什么品质？

详情见三月的狮子中的二妹！！我超喜欢。

21.你最伤痛的事是什么？

小时候没能和父母之间建立依恋的关系，导致我现在也没办法正常和大家建立亲密关系。

22.你最看重朋友的什么特点？

不以自我为中心。

23.你这一生中最爱的人或东西是什么？

西瓜。喜欢了20年了，应该也会一直喜欢下去吧。

24.你希望以什么样的方式死去？

就很安静地平静地死去呀，最好不带任何遗憾啦。

25.何时何地让你感觉到最快乐？

做出有意义的工作，自己能力得到认可的时候最快乐。

26.如果你可以改变你的家庭一件事，那会是什么？

就很希望家庭氛围能好一点，我现在不爱说话不会说话其实就是因为小时候爸妈完全不会和我沟通间接导致的吧，小时候从来不会被问做错了的事情是怎么想的，也不会问自己真正的想法，因为某些特殊原因，小学初中高中也没办法和大家一起吃饭，是从什么时候开始自己一个人呆在自己的世界里了的呢，虽然一方面说着这样不好，可现在呢，一旦有人想闯入我自己的世界将我强行拉出的时候我依然还是排斥得不行，我在这个世界里失去了很多东西，但这个世界也同样给了我很多的保护吧。

27.如果你能选择的话，你希望让什么重现？

历史历史历史，想重现历史～

28.你的座右铭是什么？

To be hungry, to be foolish.

最后又到了2019年中旬的时候啦，这一年已经过去了一半了，感觉时间过的也太快了吧，果然人越长大时间流逝得也就越快呀，感觉真奇妙，现在如果要想追上时间的脚步，就得一整天忙个不行才可以，今年的心态与去年相比真的有了很大的改进了呢，或许是因为我上半年心态不太好的原因吧。

到目前为止，今年我最大的感悟就是我的身体太差了，前段时间的考试周，先是流感，随后是胃病，最后是大姨妈，导致我连续一周瘫痪在床，又花了一周才恢复一些精气神，现在又是感冒中，嗓子都哑了，今年在身体上真的是太很多灾多难了。
